我的一些目前可以见人的项目（持续更新中）：

  • CoCiter：一个基于文献帮助定量基因和功能之间的网络服务，基于 Linux, Apache, Mysql, Python(Django) 和 Lucene。与 QIAO Nan 合作完成，半年一更，现在是 V1.2 了。即将和 CHEN Xingwei 迎来 V2~

  • DAVID 服务接口的 Python 连接 ：DAVID 是个基因注释的好网站，我们都在用它。这个练手 SOAP 的代码能被官方采用，很开心啊，第一次体会到开源贡献的成就感。不过大家注意设置延时，否则 IP 被 block 掉可不好。

  • Stepic Bioinformatics Algorithms 课的所有题解 ：Stepic 是个好网站，和 Rosalind 有千丝万缕的基情。Stepic 的题目基本都囊括到 Rosalind 里了。但是个人还是喜欢 Stepic 多些，因为课程讲的很系统。非常适合初次接触生物信息学算法的小盆友们。基于 honor code 我就不贴链接了。

  • 左袋创意网页排版 ：这个是承蒙基友委托的一个小活。趁放假帮他们做了一下前端排版。当时很神奇地被要求用静态页面，所以就没有使用 WodPress，现在想想当时真是 naive。不过也算补充了点 css 和 js 的知识。看上去他们公司很不错，flash 和图片素材都原创，赞一个。不过前端排版啥的，真心是个琐碎的方向，多分辨率兼容性和美观不可兼得啊。
