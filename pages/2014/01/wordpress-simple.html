简单WordPress建站笔记

2014年01月27日, 周一 web

按：这是之前挂在 WordPress 站上的日志。现已加入 Github Pages + Pelican。不过还是整理一下，当作纪念吧。

一年来趁着闲的时候偶尔搞了搞网站。但反反复复总会疏漏一些细节。现在小归纳一下，以供参考。

域名和主机之获取

loz4 目前是在 Namecheap 注册的域名，Hawkhost 的一年吊丝主机，用得还算顺畅。Namecheap 比较推荐，基本逢节假日会有 1 刀的优惠，不过需要信用卡或 paypal。Hawkhost，根据其 logo 俗称老鹰主机，基本也算比较适合新手上路的价格。

具体的注册购买方法网上已经有很多介绍，之后有时间再写。注意一般先有了域名然后去注册主机，否则之后还需要重新发信更改默认域名（primary domain）。

域名和主机之在一起

主要依靠的是 DNS。这里推荐使用国产 DNSPOD。
1. 在 Namecheap 里进入域名管理，点击左侧 "General" 下的 "Domain Name Server Setup"，选择 "Specify Custom DNS Servers ( Your own DNS Servers )", 加入 f1g1ns1.dnspod.net 和 f1g1ns2.dnspod.net。
2. 确保 Hawkhost 的默认域名，找到购买时发的带有 ip 地址的邮件。记下 ip
3. 去 DNSPOD 添加域名，保留两条指向 f1g1ns1.dnspod.net 和 f1g1ns2.dnspod.net 的 NS 记录（好像也删不掉）。然后再加两个 A 记录，分别指向 @ 和 www，记录值是 Hawkhost 提供的 ip。此间可能会有一阵子异常，毕竟 DNS 更新需要时间，但一般半天到一天后无论访问 http://www.loz4.com 还是 http://loz4.com 都应该顺畅无压力了。等待 DNS 更新也别闲着，可以直接上 WordPress 了。

WordPress 和 cPanel

Hawkhost 以及大部分主机商都基于傻瓜型的 cPanel 控制面板，所以图文教程可以参考 胡戈戈的中文 cPanel 安装 wordpress 教程 。
